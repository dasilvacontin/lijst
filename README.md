# lijst
JavaScript doubly linked list implementation

Current API status:
```bash
  lijst
    Capacity
      ✓ #empty()
      ✓ #length
    Element access
      ✓ #front()
      ✓ #back()
    Modifiers
      ✓ #assign()
      ✓ #pushFront()
      ✓ #popFront()
      ✓ #push() #pushBack()
      ✓ #pop() #popBack()
      - #insert()
      ✓ #erase()
      - #swap()
      - #resize()
      ✓ #clear()
    Operations
      - #splice()
      - #remove()
      - #removeIf()
      - #unique()
      - #merge()
      - #sort()
      - #reverse()
```
